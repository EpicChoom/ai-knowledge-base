# Local LLMs 🏠

Running AI models on your own hardware.

## Tools

| Tool | Type | Link | Notes |
|------|------|------|-------|
| Ollama | Local runner | https://ollama.com | Easiest way to run LLMs locally |
| LM Studio | Desktop GUI | https://lmstudio.ai | User-friendly interface |
| Unsloth | Fine-tuning | https://github.com/unslothai/unsloth | Efficient model training |

## Models to Try

| Model | Size | Link | Best For |
|-------|------|------|----------|
| Qwen3.6-27B | 27B | https://huggingface.co/unsloth/Qwen3.6-27B-GGUF | General purpose |
| Qwen3.6-35B-A3B | 35B | https://huggingface.co/unsloth/Qwen3.6-35B-A3B-GGUF | MoE architecture |
| DeepSeek Coder | Various | https://platform.deepseek.com | Code generation |

## Hardware Considerations

- **GPU**: NVIDIA cards work best (CUDA)
- **RAM**: 16GB+ recommended for larger models
- **Storage**: SSD highly recommended for model loading
